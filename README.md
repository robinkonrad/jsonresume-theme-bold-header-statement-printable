# Bold Header Statement

Magazine-inspired layout with commanding header and confident personal brand aesthetic in a more printable sizing.

**Tags:** bold, header, magazine, personal-brand, creative, modern, printable

## Use it

NEEDS CONTENT

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-bold-header-statement-printable
resume export resume.html --theme bold-header-statement-printable
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.
